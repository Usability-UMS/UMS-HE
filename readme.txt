Requirements
=============
1) Window Operating System
2) PHP 5.3.0 below


QUICK INSTALL
=============

Here is a basic outline of the installation process,
which normally takes only a few minutes:

1) Move the ums files into your web directory.

2) Restore the database using the ums.sql files provided.

3) Visit your ums site with a browser, you should
   be taken to the login page.

4) Login and start using it.
   username = admin@hotmail.com
   password = admin
   security code = Administrator

For more information, see the INSTALLATION GUIDE AND USER GUIDE:

http://usability.mimos.my/ums/solution.html

Good luck and have fun!
